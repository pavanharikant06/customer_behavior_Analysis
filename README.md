# customer_behavior_Analysis
This project focuses on analyzing consumer shopping behavior to help a retail business understand purchasing patterns, customer preferences. The analysis combines Python, SQL, and Power BI to deliver actionable, data-driven insights for improving customer engagement, marketing effectiveness, and product strategy.
